\section{何?}

文字列処理やリスト処理等，一般的によく使われる構造体や関数群を
実装したライブラリです．
いまどきはSTLやglibといった優れたライブラリが提供される環境も多い
ですから，C++が使えるならばそちらを利用した方が理に適っています
(C++でCUREを利用することも可能ではあります)．
CUREを使うのは，

\begin{itemize}
\item{C言語しか使えない}
\item{リソースが乏しい}
\end{itemize}

という時代遅れな環境での開発を余儀無くされる場合においてでしょう．
もっとも，私自身にはコンシステントなライブラリ群を構築するという
目的もあるのですが，これは私以外の人にはどうでもいい理由だと思います．
